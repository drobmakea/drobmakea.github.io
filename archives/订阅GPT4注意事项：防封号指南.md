## 3. 为什么会被封号？

以下是常见的封号原因：

- **批量注册和API滥用**：使用脚本批量注册的账户，尤其是用于 API 调用的账户，容易被封。
- **非法支付方式**：通过代付或使用被认为是盗刷的信用卡（黑卡）支付，风险极高。
- **邮箱问题**：使用乱码邮箱名或特定邮箱服务商（如 Outlook/Hotmail）的账号，封号概率较高。
- **IP地址和地理位置**：使用某些地区的 IP 地址注册或访问服务，可能被判定为批量注册。
- **支付行为异常**：频繁更换支付卡片或支付行为异常的账户，也容易被封。
